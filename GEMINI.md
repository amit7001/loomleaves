# Project Overview

This is a React web application called "Loomleaves" built with TypeScript and Vite. It serves as a simple e-commerce front-end for a textile and fabric business. The application features a home page, product listing page, product detail page, and a contact page. It uses `react-router-dom` for navigation.

## Building and Running

### Development

To start the development server, run:

```bash
npm run dev
```

### Build

To create a production build, run:

```bash
npm run build
```

### Preview

To preview the production build, run:

```bash
npm run preview
```

## Development Conventions

The project follows standard React and TypeScript conventions. The code is organized into components, pages, and a `data` directory for product information. Styling is done using CSS files that are imported into the components. The project uses a consistent visual theme across all pages.
